DogWatch:

A prediction of San Francisco Crime

[Spatial Analytics] [Python] [Machine Learning] [Folium] [Scikit-Learn]







I joined a Kaggle Competition with one other group member to find out if we coud accurately predict the type of crime that would occur in San Francisco. This work also served as the final project for my Exploratory Spatial Analytics course.

Given the location, date, and time of day of a crime that occured in the past, we created new features using grouping techniques and information from outside data sources. We applied a variety of machine learning algorithms to validate our scores. The plan for final submission will be a Flask web application that gives the user control over date and time.
